# Seven-Day Portfolio Draft Plan

Use this for a fast first draft or rescue plan.

## Day 1: Target

- Pick the target role, level, company type, and 3 skills to prove.
- Write the 3-minute reviewer takeaway.
- List all possible projects.

## Day 2: Selection

- Score projects against target relevance, ownership, process, and results.
- Choose 2-4 cases.
- Decide the order.
- Identify missing evidence for each case.

## Day 3: Homepage

- Draft role positioning.
- Write featured case cards.
- Add credibility signals.
- Make contact obvious.

## Day 4: Case Study 1

- Draft the strongest case using C2P-R.
- Add project metadata, key visuals, and result.
- Remove generic process filler.

## Day 5: Case Study 2

- Draft the second case using C2P-R.
- Make sure it proves a different useful skill than Case Study 1.
- Add honest constraints, tradeoffs, and outcomes.

## Day 6: Polish and Format

- Tighten writing.
- Check mobile and desktop readability if it is a website.
- Check PDF/deck flow if it is a presentation.
- Remove weak projects and redundant visuals.

## Day 7: Hiring Review

- Run the 20-second scan.
- Run the audit checklist.
- Rewrite the homepage and case openings.
- Prepare a short interview walkthrough for each case.
